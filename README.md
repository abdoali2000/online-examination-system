# Online Examination System

An interactive web-based examination system built using JavaScript (ES6), focusing on performance and user experience.

## 🚀 Features

- Real-time countdown timer with dynamic progress bar  
- Ability to mark and revisit questions  
- Smooth UI animations and transitions  
- Form validation for user inputs  
- Image upload functionality  
- Automatic submission when time is up  
- Instant result calculation and display  

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)  
- DOM Manipulation  
- Animate.css (for animations)
